Programs

Below are some interactive programs I've written, tending toward physics simulations, math/geometry demos, and simple games.
I started out making Java applets, but they've gone out of favor (security settings on most browsers won't let you run applets -- I only have luck with Internet Explorer), so I learned some Javascript and HTML5 to make new programs that should work on both full computers and mobile devices.
